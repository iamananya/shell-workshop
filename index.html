What is a Shell?

  • Shell is a program that let’s you run shell commands.​
  • A Shell is a command line interpreter, which runs your shell scripts.​
    • Shell is not equal to a Terminal.​
  • Examples:​ tcsh​
    fish
    ​ Bourne - Again Shell (bash)
    ​ zsh etc.​
    • Type $cat /etc/shells to display the shells available in your system

Linux Directory

  • /bin :User Binary files
  • /boot : Boot loader files
  • /dev:device files
  • /lib : system libraries
  • /tmp : Temporary Files
  • /var :variable files .... and a lot more
